## 🚀 Featured Engineering Projects

### 🏗️ 1. Integrated Smart Logistics Elevator (Weight + Distance)
*A modular elevator system with high-precision sensing and safety protocols.*
- **Key Tech:** HX711 (Load Cell/Scale), HC-SR04 (Ultrasonic), L298N Driver.
- **Innovation:** Implemented a **Safety Interlock** mechanism that halts all movement if the payload exceeds 100kg. Integrates digital signal processing for stable mass measurement.
- [View Project](./Smart-Logistics-Elevator-System)

### 🏎️ 2. RC BLDC Propulsion System
*Remote-controlled vehicle with dynamic acceleration profiling.*
- **Key Tech:** ZS-X11H ESC, BLDC Motor, PWM Ramping Logic, 4-Button UI.
- **Innovation:** Software-based "Soft Start" algorithm to manage high-torque brushless motor inertia and protect the drivetrain.
- [View Project](./RC-Brushless-Car)

### 🌐 3. StepHMI: Python-Integrated Stepper Control
*A cross-platform HMI bridging high-level software with embedded hardware.*
- **Key Tech:** Python (NiceGUI), PySerial, AccelStepper, Arduino.
- **Innovation:** Bi-directional telemetry loop for real-time position tracking through a responsive web-based dashboard.
- [View Project](./StepperControllerWithNiceGUI)

### 🏎️ 4. Autonomous 5-Channel Line-Follower
*High-precision mobile robot for complex track navigation.*
- **Key Tech:** 5-Channel IR Sensor Array, Differential Drive Logic, PWM Modulation.
- **Innovation:** High-resolution spatial feedback processing for smooth navigation of sharp turns and intersections.
- [View Project](./5-Channel-Line-Follower)
