## 🚀 Featured Engineering Projects

### 🏗️ 1. Integrated Smart Logistics Elevator
*A modular elevator system with high-precision sensing and safety protocols.*
- **Key Tech:** HX711 (Load Cell), HC-SR04 (Ultrasonic), L298N Driver.
- **Innovation:** Implemented a **Safety Interlock** mechanism that halts all movement if the payload exceeds 100kg.
- **HMI:** Real-time feedback via 16x2 I2C LCD and 3-stage visual/audible alerts.
- [View Project Details](./Smart-Logistics-Elevator-System)

### 🌐 2. StepHMI: Python-Integrated Stepper Control
*A cross-platform HMI bridging high-level software with embedded hardware.*
- **Key Tech:** Python (NiceGUI), PySerial, AccelStepper, Arduino.
- **Innovation:** Bi-directional telemetry loop for real-time position tracking through a web-based dashboard.
- [View Project Details](./StepperControllerWithNiceGUI)

### 🏎️ 3. Autonomous 5-Channel Line-Follower
*High-precision mobile robot for complex track navigation.*
- **Key Tech:** 5-Channel IR Sensor Array, Differential Drive, PWM Speed Modulation.
- **Innovation:** High-resolution spatial feedback processing for smooth navigation of sharp turns.
- [View Project Details](./5-Channel-Line-Follower)

### ⚡ 4. BLDC Motor Performance Profiler
*Power electronics testbed for high-torque brushless DC motors.*
- **Key Tech:** ZS-X11H Controller, Hoverboard BLDC Motor, PWM Ramping.
- **Innovation:** Implemented "Soft Start" routines to optimize motor longevity and power efficiency.
- [View Project Details](./BLDC-Motor-Test)

### ⚖️ 5. Precision Digital Weighing System
*Industrial-grade instrumentation and signal processing.*
- **Key Tech:** 24-bit ADC (HX711), Strain Gauge Load Cell, Signal Averaging.
- **Innovation:** Software-based calibration and digital filtering for stable mass measurement.
- [View Project Details](./Digital-Scale)
